Select the runtime-patterns-demo.bin file in the repository file list on the left.

Click the download icon at the right side of the screen to download the file to your downloads directory.

Plug the STM32F746G-DISCO board into your Mac, Linux, or Windows PC.  A USB drive labled DIS_F746NG will appear on you computer.

Drag the runtime-patterns-demo.bin file from your downloads directory to the DIS_F746NG drive.

The program should appear and begin to run on your DISCO board.
